## 🚀 How to Use the Integration

### Basic Usage (Already Active)

Your agents automatically have ECC resources:

```markdown
User → Cooper: "Build a login endpoint with OAuth"

Cooper automatically:
1. Delegates to Sage with @architect, @api-design
2. Sage designs with ECC patterns
3. Delegates to Forge with @tdd-workflow, @verification-loop
4. Forge implements with TDD, hooks auto-format
5. Delegates to Vigil with @security-scan, @code-review
6. Vigil runs security scan, checks coverage
7. Returns production-ready code ✅
```

### Advanced Usage

**Enable hooks:**
```bash
export ECC_HOOK_PROFILE=standard
node scripts/rebuild.js --force
```

**Configure MCP servers:**
```bash
# Edit agent's .claude.json
cp ecc-resources/mcp-configs/mcp-servers.json ~/.openclaw/agents/forge/.claude.json
# Add your API keys
```

**Use contexts:**
```markdown
TO: sage
CONTEXT: @research
TASK: Investigate authentication libraries

Sage loads research.md context automatically
```

**Apply language rules:**
```json
{
  "forge": {
    "rules": ["python", "common"],
    "enforce": true
  }
}
```
